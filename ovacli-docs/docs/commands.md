# Commands

```yaml
- ovacli init <path> # Initialization repository (default: current directory)
- ovacli index <path> # index videos (default: current directory)
- ovacli cook <path> # cook videos (default: current directory)
- ovacli purge <path> # purge videos (default: current directory)
- ovacli serve <repo-path> # serve videos
- ovacli version # show version
- ovacli configs # show configs
- ovacli configs default # get default config template
- ovacli configs reset # reset configs
- ovacli tsconverter <path> # convert videos to TypeScript
- ovacli ssl generate-ca # generate SSL CA
- ovacli ssl generate-cert # generate SSL certificate
```
