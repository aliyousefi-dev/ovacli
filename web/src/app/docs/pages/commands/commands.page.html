All Commands

Here you can find a list of all available commands for the ova CLI. Each command is described below, along with its usage and a code example. Use these commands to initialize, manage, process, and serve your video repositories efficiently. For more details on each command, refer to the corresponding section and code preview. If you need further assistance, consult the official documentation or use the --help flag with any command.

Initialization Command

Initializes the ova CLI in the specified path.

Index Command

Indexes the video files.

Cook Command

Processes and prepares the video files.

Purge Command

Removes all indexed and cooked data.

Serve Command

Serves the repository at the specified path.

Version Command

Displays the current version of ova CLI.

Configs Command

Shows the current configuration.

Configs Default Command

Resets configuration to default values.

Configs Reset Command

Resets all configuration.

TS Converter Command

Converts files using the tsconverter tool.

SSL Generate CA Command

Generates a Certificate Authority for SSL.

SSL Generate Cert Command

Generates an SSL certificate.
